## 一、自我决定理论（SDT）

### 三种基本心理需求

```
自主性（Autonomy）：
  → "我有选择权" "这是我自己决定的"
  → 不是放任自流，而是在限定框架内提供选择

胜任感（Competence）：
  → "我能成功" "我擅长这个"
  → 不是降低难度，而是在最近发展区内成功

归属感（Relatedness）：
  → "我被接纳" "我是团体的一部分"
  → 不是消除竞争，而是确保每个人被看到
```

### 三者与学习效果的关系

```
自主性 + 胜任感 + 归属感 = 内在动机 → 深度参与 → 持久学习

缺乏任何一个，都会导致动机减弱：
  缺少自主性 → 被动应付
  缺少胜任感 → 焦虑逃避
  缺少归属感 → 孤独放弃
```

## 二、课堂中的三元设计策略

### 自主性设计

```
错误做法："这次作业必须完成X题"（无选择）
正确做法：
  - 任务选择："你可以选择A、B、C三个任务之一"
  - 方法选择："你可以用文字/图表/展示来解释"
  - 顺序选择："先做基础练习还是先尝试挑战题？"
  - 小组选择："自己组队还是老师分组？"
  
⚠️ 注意：选择必须都通往同一个教学目标
    不能为了选择而牺牲目标达成
```

### 胜任感设计

```
错误做法：降低难度让所有人都能做对（虚假胜任感）
正确做法：
  - 三级难度：「人人能起步，人人有挑战」
  - 明确标准：「满分答案长什么样，大家一开始就知道」
  - 即时反馈：「做对了立刻知道，做错了立刻知道错在哪」
  - 进步可见：「让学生看到"上次我不会，这次我会了"」
  
⚠️ 注意：胜任感来自真实能力的提升，不是来自虚假的简单
    所以要设计"刚刚够得着"的挑战
```

### 归属感设计

```
错误做法：分组后各做各的（物理一起，心理独立）
正确做法：
  - 每个人有不可替代的角色
  - 展示环节确保每个人被看到
  - 同伴互评：不是挑毛病，而是"我发现你一个优点"
  - 集体建构：全班合作完成一个最终作品（都有署名）
```

## 三、关于动机的两个反直觉发现

### 发现1：动机是能力的副产品

> 不是有了兴趣才去学习，而是学会了才有兴趣。
> 真正的内驱力来源于胜任感。

```
传统的因果链：兴趣 → 学习 → 能力
真实的因果链：能力（小成功）→ 胜任感 → 兴趣 → 更多学习

启示：不要先追求"有趣"，先让学生体验"我能行"
  → 降低第一道门槛（让所有人成功一步）
  → 再逐步提高难度（让成功感驱动继续）
```

### 发现2：有效失败的价值

> 认知负荷理论倾向于给最顺滑的学习路径，
> 但某些"困难"对长期学习反而有益。

```
有效失败（Productive Failure）：
  → 在教正式解法之前，让学生先尝试解决
  → 可能失败，但失败本身就是学习
  → 短期增加认知负荷，长期提升理解深度

教学设计应用：
  - 在探究课中保留适度的"挣扎空间"
  - 不要急于给出答案，先让学生暴露困惑
  - "混乱"恰恰是认知冲突的表现，是有价值的
```

## 四、动机设计的差异化策略

```
基础动机策略（全体通用）：
  - 明确的成功标准（知道怎么做算"好"）
  - 即时正向反馈（做对立刻知道）
  - 安全的犯错环境（"错是学习的一部分"）

标准动机策略（大部分学生需要）：
  - 任务选择权（2-3个选项）
  - 同伴互动（互评/互相讲解）
  - 进步可视化（看到自己的成长）

拓展动机策略（学有余力）：
  - 自主设定目标
  - 创造性的表达方式
  - 真实的受众（不止是老师看）
```
