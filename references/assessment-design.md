## 一、GRASPS 真实情境任务模型

### 六要素

| 字母 | 要素 | 问题 | 示例 |
|------|------|-----|------|
| **G** | Goal 目标 | 你的任务是......目标是......挑战是...... | "你的任务是为社区设计一个节水方案" |
| **R** | Role 角色 | 你的角色是...... | "你是城市规划师" |
| **A** | Audience 受众 | 你的受众是...... | "向社区委员会汇报" |
| **S** | Situation 情境 | 情境是...... | "该社区正面临十年一遇的干旱" |
| **P** | Product 成果 | 你需要交付...... | "一份方案报告+预算表" |
| **S** | Standards 标准 | 成功的标准是...... | "方案可行性>80%/预算不超过X元/至少包含3种策略" |

### 自检三问

```
1. 这个任务在真实世界中是否有可能发生？(不是"看起来真实"的伪情境)
2. 完成这个任务是否必须用到本课的知识？(知识是完成任务不可绕过的工具)
3. 这个任务是否有多种解决方案？(不是只有一个标准答案的表演)
```

## 二、SOLO分类评价

### 五个层次

| 层次 | 定义 | 评分描述 |
|------|------|---------|
| 前结构 (P) | 没有理解，答非所问 | 1分 |
| 单点结构 (U) | 只提到一个相关方面 | 2分 |
| 多点结构 (M) | 提到多个方面，但无联系 | 3分 |
| 关联结构 (R) | 多个方面整合成整体 | 4分 |
| 拓展抽象 (E) | 迁移到新情境/提出新问题 | 5分 |

### 评价量规模板

```
[任务描述]
评分标准：
  P(1分): [具体表现]
  U(2分): [具体表现]
  M(3分): [具体表现]
  R(4分): [具体表现]
  E(5分): [具体表现]
```

## 三、AI评估三分类

> 来源：数字教育委员会与Pearson报告（AIED知识库）

### 三类评估定义

| 类别 | 定义 | 设计原则 | 示例 |
|------|------|---------|------|
| **无AI** | 学生独立完成，禁止使用AI | 在监控条件下进行 | 课堂限时测验、口试、现场写作 |
| **AI辅助** | 学生可用AI，但需声明 | 提交AI使用声明+个人修改记录 | 在家完成的作文（标明AI参与部分） |
| **AI整合** | AI是学习环境的一部分 | 评估人机协作能力 | 用AI分析数据+人工解读结果 |

### AI韧性设计14法（精选）

```
1. 过程记录法：要求学生记录完成任务的每一步（含AI使用）
2. 作品+答辩法：提交作品后口头答辩（考察真实理解）
3. 情境化任务：采用GRASPS式真实任务（AI无法套模板）
4. 多模态证据：文字+图表+口头+实操（单一证据不可靠）
5. 迭代追踪：比较初稿和终稿的变化（可见学生的思考过程）
6. 反思声明：要求学生反思"AI帮了我什么""我还需要学什么"
7. 协作评估：团队完成+个人贡献自评+同伴互评
```

### AI使用声明模板

```
我在本次作业中使用了AI工具：[工具名称]
AI在以下环节提供了帮助：[列出具体环节，如"帮我理清思路""生成初稿"]
我的独立修改/补充：[列出我做了哪些AI之外的工作]
这次经历让我意识到我还需要学习：[自我反思]
```

## 四、四层诊断测验原理

> 来源：Šmida et al. (2026) 四层诊断测验研究（AIED知识库）

### 四层结构

```
第一层：答案选择
  → 单选/多选（检测知识）
  
第二层：理由选择
  → 为什么选这个答案（检测逻辑）
  
第三层：答案信心
  → 你对这个答案有多确定？（确定/不确定）
  
第四层：理由信心
  → 你对这个理由有多确定？（确定/不确定）
```

### 判分标准

```
真正掌握 = 第一层正确 + 第二层正确 + 第三层确定 + 第四层确定
缺一不可！

诊断价值：
  - 答案对+理由对+都确定 = 真正掌握 ✅
  - 答案对+理由对+不确定 = 部分掌握（需巩固）
  - 答案错+理由错+都确定 = 根深蒂固的迷思概念 ⚠️
  - 答案错+理由错+不确定 = 知识缺乏（补知识就行）
```

### 课堂简化版

日常课堂不需要完整的四层诊断，可用简化版：

```
两步检测：
  Step 1: 学生写出答案
  Step 2: 学生在旁边写"我有多确定？"（1-5分）
  
  教师看什么：
    - 高分+高确定 → 真正会了
    - 高分+低确定 → 蒙对的（需要同类题再测）
    - 低分+高确定 → 迷思概念（需要追溯源头纠正）
```
