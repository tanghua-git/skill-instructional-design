## 一、迷思概念 vs 知识缺乏

### 关键区分

```
知识缺乏：学生不知道 → 教学对策：讲清楚/给信息/示范
迷思概念：学生坚信一个错误的理解 → 教学对策：先动摇旧观念，再重建

如何区分？
  - 四层诊断：选错+理由错+很确定 = 迷思概念
  - 四层诊断：选错+理由错+不确定 = 知识缺乏
```

## 二、认知冲突设计

### 建构主义核心

> 学习不是信息的灌输，而是图式的构建。
> —— 建构主义学习理论

### 认知冲突三步法

```
Step 1：激活旧图式
  → 先让学生用自己的已有知识解释一个新现象
  → 暴露当前理解（无论对不对）

Step 2：制造冲突
  → 展示一个旧图式无法解释的证据/现象
  → 关键：学生必须亲自感到"我的理解不够用了"

Step 3：顺应/同化
  → 在认知需求驱动下，学生主动建构新理解
  → 教学提供脚手架帮助学生重建图式
```

### 认知冲突设计示例

| 学科 | 旧图式（学生以为） | 冲突事件 | 新图式（教学目标） |
|------|------------------|---------|-----------------|
| 科学 | 金属是固体 | 汞是金属但液态 | 物质状态不是金属的唯一特征 |
| 数学 | 乘法结果一定比乘数大 | 0.5×8=4（比8小） | 乘数<1时结果变小 |
| 语文 | "方便"就是便利 | "你方便时来" vs "你在我方便时来" | 语境决定语义 |
| 历史 | 落后就要挨打 | 为什么当时最强大的是落后民族入侵中原 | 文明兴衰的复杂因素 |

## 三、学情分析的四个维度

```
1. 知识维度
   - 学生已经学过什么？（前驱知识）
   - 他们掌握到什么程度？
   - 这个新知识应该"挂"在哪个旧知识上？（图式理论）

2. 认知维度
   - 学生的认知水平处于什么阶段？
   - 抽象思维能力如何？
   - 信息加工能力如何？（认知负荷容量）

3. 误解维度
   - 学生可能有什么迷思概念？
   - 这些误解的源头在哪里？（日常经验/之前教学的简化）
   - 哪些误解最顽固？

4. 动机维度
   - 学生对什么感兴趣？
   - 学习内容与他们的生活有什么关系？
   - 什么能让她们觉得"值得学"？
```

## 四、通用迷思概念类型库

| 类型 | 特征 | 常见领域 | 教学对策 |
|------|------|---------|---------|
| 过度简化 | "只要X就能Y" | 所有学科 | 提供反例 |
| 线性因果 | 以为只有一个原因 | 科学/历史 | 展示多因素模型 |
| 本质主义 | 以为"本质"不可改变 | 生物/化学 | 展示连续变化 |
| 类比误用 | 用日常经验理解科学概念 | 物理/科学 | 直接指出类比局限 |
| 混淆相关与因果 | "因为A和B一起出现" | 数学/科学 | 设计对照实验 |
